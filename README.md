# エンドフィールドのパズルを解くやつ

4×4〜8×8の盤面に対して、行・列ごとの埋めマス数を満たすようにポリオミノを敷き詰めるパズルを解くWebアプリです。  
盤面エディタで条件を設定し、ピースビルダーでピースを作成して探索できます。

## 主な機能

- 盤面サイズ: 横×縦（4〜8）
- セル状態: 空 / 障害物 / 固定（カウントされる障害物）
- 行・列ごとのターゲット数入力
- ピース作成（形状、個数、回転可否、色）
- WebWorkerによる探索（UIを止めない）
- 解の表示（色付き、前/次で切り替え）

## セルの意味

- 空: ピースが置ける
- 障害物: ピースを置けない、ターゲットにカウントしない
- 固定: 既に埋まっている扱い。ターゲットにカウントされ、ピースは置けない

## 使い方

いつかアップロードします どこかに。

ブラウザで表示されたURLを開き、盤面・ターゲット・ピースを設定して「探索開始」を押します。

## 注意点

- 探索開始には、バリデーションが通る必要があります。
- ピースの総マス数は、ターゲット合計（固定セル分を除いた数）と一致する必要があります。

## 技術スタック

- React + TypeScript + Vite
- WebWorker（探索処理）
